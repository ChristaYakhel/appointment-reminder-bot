 APPOINTMENT REMINDER BOT

PROBLEM STATEMENT
Many people forget their appointments, leading to missed meetings or doctor visits.  
The Appointment Reminder Bot helps users schedule, view, and get reminders for their appointments through a simple chat-style interface.

FEATURES
- Schedule appointments step by step
- View all appointments
- Get reminders for tomorrow’s appointments
- Chat-style interface for easy interaction

HOW IT WORKS
1. Open the bot → Welcome message appears  
2. Type `schedule` → Bot guides you through:
   - Step 1: Choose specialty  
   - Step 2: Enter doctor name  
   - Step 3: Enter hospital name  
   - Step 4: Enter date (YYYY-MM-DD)  
   - Step 5: Enter time (HH:MM)  
3. Type `view` → Bot shows all scheduled appointments  
4. Type `reminder` → Bot shows appointments for tomorrow  

TECHNICAL STACK
- Python 3
- Tkinter for GUI
- JSON for data storage
